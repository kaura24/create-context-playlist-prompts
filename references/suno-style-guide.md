# Suno Field Guide

Use official Suno guidance as the public field-routing contract and this skill as the private project contract.

## Route Content By Field

- Styles or Basic Prompt: genre, feel, tempo, groove, vocal behavior, instruments, harmony, form, and production
- Exclude or Absolute Exclusion Prompt: unwanted genres, instruments, voice styles, performance traits, and mix traits
- Lyrics: title-independent lyric text, supported section tags, and section-specific delivery context
- Title: the song title only, rendered outside every code block

Suno Custom Mode provides distinct Lyrics, Styles, Advanced options, and Title inputs. Keep the generated artifacts separate so the user can paste them directly.

## Write Specific Audible Direction

Prefer concrete music vocabulary:

- measured BPM, meter, pocket, syncopation, or rubato
- section movement such as restrained entry, crescendo, bridge contrast, return, or coda
- instrument role, register, texture, and articulation
- vocal range, phrasing, pronunciation, vibrato, breath, and emotional depth
- mic distance, room, dynamics, stereo position, transients, tone, and processing

Use evocative language only when it implies an audible behavior. Keep reference artist and song names in analysis, not generation fields.

## Keep Tags Functional

Use supported terms according to musical function:

Intro, Verse, Pre-Chorus, Chorus, Bridge, Outro, Hook, Refrain, Break, Drop, and Coda.

Keep bracketed tags in Lyrics, not Styles. Use a literal supported sequence when a genre-specific form name does not map cleanly. Never invent poetic section tags.

## Apply Project Limits

Use 800 characters for the Basic Prompt and 100 characters for Absolute Exclusion as conservative project limits. Do not present them as published Suno platform maxima.

Current Suno documentation states that recent models can generate longer than four minutes, but this project intentionally requires actual results between 180 and 240 seconds. TrackSpec planning and audio metadata enforce the narrower contract.
